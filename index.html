Clase Maqueen

Maqueen es una plataforma robótica desarrollada por la compañía DFRobot. Esta plataforma está diseñada para ser fácil de programar y está dirigida principalmente a estudiantes y aficionados de robótica. El robot viene con una serie de sensores y actuadores incorporados, incluyendo LEDs, sensores de línea y un sensor de distancia. Los usuarios pueden programar el robot para que realice una variedad de tareas, como seguir una línea o evitar obstáculos. El Maqueen es compatible con varios lenguajes de programación, incluyendo Python, Arduino, y micro:bit, lo que lo hace muy versátil.

  • Maqueen


    • Funcionalidades

    1. Control de movimiento: El Maqueen puede moverse hacia adelante, hacia atrás y girar.
    2. Sensor de distancia: El robot viene con un sensor de distancia HC-SR04 incorporado para medir la distancia a objetos cercanos.
    3. Sensores de línea: El Maqueen cuenta con dos sensores de línea para seguir una línea o detectar un borde.
    4. LEDs: El robot viene con dos LEDs frontales que se pueden encender o apagar.
    5. Sonido: El robot puede reproducir sonidos y música.
    6. Programación: El Maqueen scon micro:bit se puede programar en varios lenguajes, incluyendo Python, Arduino y por bloques.

    Funciones disponibles de maqueen

    1. set_led

      Este método se utiliza para encender o apagar los LEDs frontales del robot. Recibe dos parámetros: el número del LED (0 para el izquierdo y 1 para el derecho) y el valor (1 para encenderlo y 0 para apagarlo).

      
robot = Maqueen()
robot.set_led(0, 1)  // Encender el LED izquierdo
robot.set_led(1, 0)  // Apagar el LED derecho

    2. read_distance

      Este método lee la distancia desde un sensor HC-SR04. El resultado se devuelve en centímetros.

      
robot = Maqueen()
distancia = robot.read_distance()
print("La distancia es:", distancia, "cm")

    3. read_patrol

      Este método lee el estado del sensor de patrulla. Recibe un parámetro que indica qué sensor leer (0 para el izquierdo y 1 para el derecho). Devuelve 1 si el sensor detecta una línea negra, y 0 en caso contrario.

      
robot = Maqueen()
left_sensor = robot.read_patrol(0)
right_sensor = robot.read_patrol(1)
print("Sensor izquierdo:", left_sensor, "Sensor derecho:", right_sensor)

    Sigue Líneas con Maqueen

    Los sensores son dispositivos que detectan algún tipo de cambio en su entorno y convierten esa información en señales eléctricas que pueden ser procesadas por una computadora o un robot. En el caso del seguimiento de líneas con Maqueen, los sensores son sensores infrarrojos.

    Sensores Infrarrojos

    Los sensores infrarrojos son dispositivos que emiten y reciben luz infrarroja. Los sensores infrarrojos utilizados en el seguimiento de líneas del Maqueen constan de dos partes: un emisor y un receptor. El emisor emite una señal de luz infrarroja que se refleja en la superficie del suelo y es detectada por el receptor.

    Cuando el robot se mueve sobre una línea negra, la luz infrarroja es absorbida por la línea y no se refleja hacia el receptor. Esto provoca una disminución en la señal de salida del receptor. Por otro lado, cuando el robot se mueve sobre una superficie clara, la luz infrarroja se refleja hacia el receptor, lo que resulta en una señal de salida más fuerte.

    Sigue Líneas con Maqueen

Si el sensor izquierdo detecta la línea negra, el robot se moverá hacia la derecha para seguir la línea. Si el sensor derecho detecta la línea negra, el robot se moverá hacia la izquierda para seguir la línea. Si ambos sensores detectan la línea negra, el robot continuará recto.
